Privacy Policy for Hint – Quick Notes. No Pen.

Hint – Quick Notes. No Pen. does not collect, store, or share any personal user data.

This extension only accesses the text that the user explicitly selects on a webpage when they choose the “Add to Notes” option from the context menu.

The selected text, notes, and collections are saved locally inside the user’s browser using Chrome storage and are never transmitted to any server.

What we do NOT do

  • We do not track browsing history
  • We do not monitor user activity
  • We do not collect personal information
  • We do not share any data with third parties

All notes, collections, and saved content remain stored only on the user’s device.

Permissions Explanation

activeTab

The activeTab permission is used only when the user explicitly selects text on a webpage and chooses “Add to Notes” from the context menu. This temporary access allows the extension to read the selected text from the current tab and save it into the user’s notes. The extension does not access any webpage data unless the user performs this action.

scripting

The scripting permission is used to run a small script on the active webpage to capture the text that the user has selected. This script runs only after the user clicks “Add to Notes” from the context menu and is required to retrieve the selected content from the page.

sidePanel

The sidePanel permission is used to open the notes editor inside Chrome’s side panel when the user selects text from a webpage. In the side panel, users can preview and edit the selected text before saving it, add it to an existing note, or create a new note and collection. The side panel also displays the list of previously created notes and collections and provides a button to navigate to the main dashboard. From the editor, users can manage their notes by editing, organizing, deleting, and downloading them as PDF, Word, or text files.

storage

The storage permission is used to save notes, collections, and user content locally within the browser.

User Control

Users have full control over their data. They can edit, delete, organize, and download their notes at any time as PDF, Word, or text files.
